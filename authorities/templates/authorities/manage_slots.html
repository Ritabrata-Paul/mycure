Time Slots Guide

Time slots define when your services are available for booking. Keep these points in mind:

  • Create separate time slots for different services
  • Assign specific doctors to slots where applicable
  • Set the capacity based on how many appointments you can handle simultaneously
  • You can add multiple slots for the same day with different time ranges
